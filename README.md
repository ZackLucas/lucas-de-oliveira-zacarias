## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Run on Docker

```bash
# build image docker without cache
$ docker-compose build --no-cache

# run docker
$ docker-compose up
```
